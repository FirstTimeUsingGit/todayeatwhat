# 今晚食乜餸 - 智能晚餐食譜生成器

## 功能特色

1. **季節性材料推薦** - 根據當前季節推薦適合的食材
2. **多樣化菜系選擇** - 包括中式、西式、日式、韓式、泰式、台式等
3. **菜式類別篩選** - 主菜、湯水、甜品、健康餐、重口味等
4. **食材圖片上傳** - 支持上傳食材圖片以便識別
5. **食譜數據庫** - 包含多種時令食譜
6. **icook.tw 數據同步** - 定期載入最新食譜
7. **自定義食譜** - 支持用戶添加自己的食譜

## 技術架構

- **前端**: HTML, CSS, JavaScript
- **後端**: Python Flask
- **數據來源**: icook.tw 食譜數據

## 功能詳述

### 1. 季節性材料推薦
- 春季：筍、韭菜、菠菜、豌豆苗、小蔥、香椿
- 夏季：冬瓜、苦瓜、絲瓜、茄子、空心菜、豆角
- 秋季：南瓜、蓮藕、栗子、白蘿蔔、白菜、柿子
- 冬季：白菜、蘿蔔、羊肉、黑木耳、銀耳、桂圓

### 2. 食材圖片上傳
- 支援圖片拖拽上傳
- 即時預覽功能

### 3. 食譜生成邏輯
- 根據選擇的季節、菜系、類別篩選食譜
- 智能匹配最適合的食譜

### 4. 食譜編輯功能
- 可添加自定義材料
- 可編輯自定義食譜

### 5. 載入最新食譜
- 從 icook.tw 同步最新食譜數據

## 部署與運行

```bash
# 安裝依賴
pip install flask requests beautifulsoup4

# 運行應用
python app.py
```

應用將運行在 http://localhost:8080

## API 端點

- `GET /api/recipes` - 獲取所有食譜
- `GET /api/recipes/filter?cuisine={}&season={}&category={}` - 根據條件篩選食譜
- `POST /api/recipes/load_latest` - 載入最新食譜

## 文件結構

- `index.html` - 主頁面
- `app.py` - Flask 應用
- `scrape_icook.py` - icook.tw 數據爬蟲
- `data.json` - 食譜數據存儲

## 使用說明

1. 選擇當前季節
2. 選擇手邊的食材
3. 選擇偏好菜系
4. 選擇菜式類別
5. 點擊「生成食譜」獲取推薦
6. 如需更多食譜，可點擊「載入最新食譜」
7. 可通過「編輯材料」和「編輯食譜」添加自定義內容
